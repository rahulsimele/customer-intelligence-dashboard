
# Customer Intelligence Dashboard

A full-stack **Customer Intelligence Dashboard** built using **Django (Backend)** and **React (Frontend)**.
This project helps businesses analyze customer data and generate actionable insights.

---

# Project Overview

The Customer Intelligence Dashboard provides:
- Centralized customer data management
- Analytics-ready data models
- Interactive dashboards for business insights

---

# Tech Stack

# Backend
- Python
- Django
- Django REST Framework
- PostgreSQL

# Frontend
- React.js

# Tools
- Git & GitHub
- Virtual Environment (venv)

# Backend Setup

1. Clone the repository
2. Create and activate virtual environment
3. Install dependencies
4. Run migrations
5. Start Django server

---

# Frontend Setup

1. Navigate to frontend folder
2. Install npm packages
3. Start React app
